

直接迭代文件，with语句自动关闭文件

```python
with open ('test.txt') as f:
    for i in f: # 直接迭代文件
        # state
```



也是直接迭代的文件

```python
infile=open('test.txt')
for i in infile:
    # state
infile.close()
```







旧方法，while循环，一次读取一行


```python
infile=open('xxx.txt')
line=infile.readline().strip()
while line:
    # state
    line=infile.readline().strip() # 用while, readline，这一行一定要加不然是死循环。
infile.close() 
```
